This project addresses the challenges of campus navigation at **Obafemi Awolowo University (OAU)** by developing a **web-based Geographic Information System (WebGIS)** application. The platform enables users to search for locations, view detailed spatial information, and generate optimal routes across the university campus through an interactive and user-friendly interface.

The application integrates spatial and attribute data into a functional geodatabase and delivers fast, accurate, and intuitive navigation for students, staff, and visitors.

## ğŸ—º Data Collection & Processing
The study involved the collection, processing, and integration of spatial datasets representing key campus features, including:

- **115 Points of Interest (POIs)**  
- **1,346 Residential Units**  
- **Road Network Data**  
- **Administrative Boundaries**

All datasets were cleaned, structured, and integrated into a **geodatabase** to support spatial queries, routing, and map rendering.
